# Chalice-rium

## 概要
[Bloodborne](https://www.playstation.com/ja-jp/games/bloodborne/) の聖杯ダンジョン用メモアプリです。

### 機能

#### ダンジョン管理
- [X] ユーザーの CRUD
- [X] 聖杯ダンジョンの CRUD
- [X] ユーザーが作成した聖杯ダンジョン一覧の表示
- [ ] 聖杯ダンジョンの検索
    - [ ] 聖杯文字による検索
    - [ ] 深度によるフィルタリング
    - [ ] 区画によるフィルタリング
    - [ ] 追加儀法によるフィルタリング
    - [ ] 登録日時による並び替え表示
- [ ] 聖杯ダンジョンのお気に入り登録機能
- [ ] 聖杯ダンジョンへのコメント投稿
